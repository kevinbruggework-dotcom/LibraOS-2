📲 Installera som app
Följ stegen för din enhet:
iPhone / iPad (Safari)
1. Öppna LibraOS i Safari
2. Tryck på Dela-ikonen (ruta med pil upp) 📤
3. Scrolla och välj "Lägg till på hemskärmen"
4. Tryck "Lägg till" — klart! 🎉
Android (Chrome)
1. Öppna LibraOS i Chrome
2. Tryck på ⋮ (tre punkter) uppe till höger
3. Välj "Lägg till på startskärmen"
4. Tryck "Lägg till" — klart! 🎉
Appen fungerar offline och startar utan webbläsarens UI — precis som Signal eller Telegram.

LibraOS krypterar dina meddelanden men din IP-adress är synlig för din internetleverantör. En VPN döljer även det.
